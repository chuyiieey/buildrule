# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of BuildRule
- Core functionality including RuleNode base class and logical combinators
- Built-in rules for numeric, string, boolean, datetime, set, list, dictionary, regex, and XML operations
- RuleBuilder for creating complex rule structures
- Serialization and deserialization support
- Comprehensive test suite
- Documentation in both Chinese and English

## [0.0.1] - 2024-01-01

### Added
- First public release
- Basic rule engine functionality
- Support for custom rule creation
- Serialization and deserialization of rule expressions
- Multiple rule types covering common use cases
- English and Chinese documentation
